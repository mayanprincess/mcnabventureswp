Características

  • Responsive: Desktop (4 col), Tablet (3 col), Mobile (2 col)
  • Tabs: Filtrado por Fotos/Videos
  • Carrusel: Paginación con dots y botones anterior/siguiente
  • Accesible: ARIA labels, navegación por teclado
  • Sin dependencias: Vanilla JavaScript puro
  • Design System: Colores y tipografía de McNab Ventures

Archivos Generados

  • views/components/multimedia.twig - Plantilla Twig
  • assets/scss/components/_multimedia.scss - Estilos SCSS
  • assets/js/multimedia.js - Lógica JavaScript
  • inc/components-registry.php - Registro en ACF

Registrado en WordPress

El componente está disponible en el panel de componentes de WordPress. Accede a: Tema → Componentes → Multimedia
